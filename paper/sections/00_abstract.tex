\section{Abstract}

Fraud detection systems face challenges including class imbalance, concept drift, and adversarial evasion. This paper presents an agentic fraud detection system built on the MAPE-K (Monitor-Analyze-Plan-Execute using Knowledge) architecture with LLM-driven contextual decision-making.

The system integrates: CTGAN-based data augmentation for class imbalance, adversarial training for robustness, feature extraction with rule-based scoring, pattern learning for fraud classification (phishing, social engineering), and a LLM decision agent (Ollama Llama 3) for autonomous decision-making. The system uses a SQLite knowledge store for persistent memory with feedback-driven continuous learning.

Our experiments show: F1 of 0.9877, ROC-AUC of 0.9869, 98.34% robustness. Drift detection works with PSI=8.09 and KS=0.50. After adaptation (retraining on drifted data), the system recovers to F1=0.9929.

The MAPE-K loop detects drift via PSI/KS tests, triggers retraining with L1/L2 feedback loops, and uses a LLM (Llama 3) to make contextual decisions about deployment vs. retraining. This provides truly autonomous fraud detection that adapts to evolving fraud patterns.
